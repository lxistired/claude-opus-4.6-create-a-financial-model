# Alphabet（Google）财务模型

## 1) 建模目标与方法
- 目标：基于 2025 年 10-K 搭建可复用估值框架，产出每股内在价值区间。
- 方法：5 年显性期 + 永续增长终值的 DCF（FCFF 口径）。
- 货币单位：**百万美元**（每股价值为美元）。

## 2) 历史数据（来自 2025 年报）

### 2.1 利润与现金流（2023-2025）
- Revenue：307,394 / 350,018 / 402,836
- Operating Income：84,293 / 112,390 / 129,039
- Net Income：73,795 / 100,118 / 132,170
- CFO：101,746 / 125,299 / 164,713
- CapEx：32,251 / 52,535 / 91,447

### 2.2 关键观察
- 收入 CAGR（2023-2025）：**14.5%**
- 净利润 CAGR（2023-2025）：**33.8%**
- 经营利润率：27.4% -> 32.1% -> 32.0%
- FCF（简化为 CFO-CapEx）：69,495 -> 72,764 -> 73,266（CapEx 加速导致现金流率走低）

### 2.3 分部表现（2025）
- Google Services 收入：342,721；经营利润：139,404
- Google Cloud 收入：58,705；经营利润：13,910
- Other Bets 收入：1,537；经营亏损：-7,515
- Alphabet-level activities 经营亏损：-16,760

## 3) 关键假设（基准情景）

### 3.1 经营假设
- 收入增速（2026-2030）：14%、12%、10%、8%、6%
- EBIT Margin：32.2%、32.5%、32.7%、32.9%、33.0%
- 有效税率：18.0%（较 2025 的 16.8%小幅回归）
- D&A/Revenue：5.3%、5.2%、5.1%、5.0%、5.0%
- CapEx/Revenue：20%、19%、18%、17%、16%（AI 基建高位后逐步回落）
- ΔNWC：按收入增量的 0.5%

### 3.2 折现与终值假设
- 无风险利率（Rf）：4.20%（2/9/2026 附近美债 10Y）
- Beta：1.05（大盘科技股常见估计区间中枢，保守设定）
- 市场风险溢价（ERP）：5.0%（长期经验假设）
- 股权成本（Ke）：4.20% + 1.05*5.0% = **9.45%**
- 税前债务成本（Kd）：4.9%（参考 10-K 新发债券票息）
- 税后债务成本：4.9%*(1-18%)=**4.02%**
- 资本结构权重（E/D）：97% / 3%
- WACC：约 **9.2%**
- 永续增长率（g）：**3.0%**（2% 通胀目标 + 真实增长）

## 8) 结果解读
- 模型显示 Alphabet 的内在价值高度依赖两个变量：**长期增长率** 与 **WACC**。
- 2025 年 CapEx 显著抬升，短期压制自由现金流；若 AI 资本开支效率改善，估值弹性更大。
- Other income 波动（2025 年较高）不宜线性外推，已通过 NOPAT 框架降低一次性影响。

## 9) 已知局限（下一步可优化）
- 目前是“公司整体”建模，未分拆 Services/Cloud 的不同增速与利润率轨迹。
- 未纳入股票回购对每股价值动态摊薄/增厚的年度路径。
- 未引入监管罚款与诉讼支出的概率情景（可追加蒙特卡洛版本）。
