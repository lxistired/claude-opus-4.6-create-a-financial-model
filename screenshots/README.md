# Excel 模型截图说明

请将 `Alphabet_IB_Model_v5.xlsx` 的以下页面截图后放入本目录，便于在 GitHub 上展示模型结构。

## 建议截图清单

| 文件名建议 | 对应 Tab | 说明 |
|------------|----------|------|
| `01-cover.png` | **Cover** | 封面：公司名、模型说明、使用步骤 |
| `02-key-summary.png` | **Key_Summary** | 执行摘要：核心 KPI、分部收入、BS 亮点、投资亮点与风险 |
| `03-assumptions.png` | **Assumptions** | 假设页：情景选择器、WACC、收入增速、CapEx、历史锚点 |
| `04-segment-revenue.png` | **Segment_Revenue** | 分部收入：6 条业务线自底向上 + Revenue Mix% |
| `05-segment-pl.png` | **Segment_PL** | 分部 P&L + 交叉验证行（Difference = 0） |
| `06-consolidated-pl.png` | **Consolidated_PL** | 合并利润表：Revenue → EBIT → NI → EPS |
| `07-bs.png` | **BS** | 资产负债表 + Balance Check 行 |
| `08-cash-flow.png` | **Cash_Flow** | 现金流量表：CFO、CapEx、FCF |
| `09-dcf.png` | **DCF** | DCF 估值：UFCF、终值、EV、每股价值 |
| `10-sensitivity.png` | **Sensitivity** | 敏感性矩阵：WACC × g → 每股价值 |
| `11-ratio-analysis.png` | **Ratio_Analysis** | 比率分析：盈利、效率、杠杆、现金流质量 |

## 截图建议

- 格式：PNG 或 JPG 均可。
- 可只截取表格主体区域（含年份列与关键行），无需整屏。
- 若某 Tab 过长，可截上半部分或分两张（如 Assumptions 上/下）。

截图完成后，在本仓库 README 的「模型截图」章节引用这些图片即可。
